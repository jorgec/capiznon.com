# Capiznon Geo Theme Documentation

A modern, location-based WordPress theme featuring interactive maps for showcasing establishments, tourist destinations, and local businesses.

## Features

- **Interactive Map** - Full-screen Leaflet.js map with marker clustering
- **Location Management** - Custom post type with comprehensive fields
- **Hierarchical Categories** - Types, subtypes, areas, tags, and price ranges
- **Nearby Locations** - Find places within a configurable radius
- **REST API** - Full API for headless/decoupled usage
- **Modern Design** - Tailwind CSS for sleek, professional styling
- **Mobile Responsive** - Works beautifully on all devices
- **No Build Step** - Uses CDN for Tailwind and Leaflet (shared hosting friendly)

## Quick Start

1. Activate the theme in WordPress Admin → Appearance → Themes
2. Go to Appearance → Geo Settings to configure default map center
3. Add locations via Locations → Add New
4. Set your homepage to display the map (Settings → Reading → Static page → Front page)
