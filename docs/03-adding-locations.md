# Adding Locations

## Creating a New Location

1. Go to **Locations → Add New**
2. Enter the location title
3. Add a description in the editor
4. Fill in the meta boxes (see below)
5. Set a featured image
6. Publish

## Location Fields

### Location Details

- **Street Address**: Main address line
- **Address Line 2**: Building, floor, unit (optional)
- **City**: Default: Roxas City
- **Province**: Default: Capiz
- **Postal Code**: ZIP/postal code
- **Country**: Default: Philippines

### Map Coordinates

Click on the map to set the location, or enter coordinates manually:

- **Latitude**: e.g., 11.5853
- **Longitude**: e.g., 122.7511

**Tip**: Use Google Maps to find coordinates:
1. Right-click on a location in Google Maps
2. Click the coordinates to copy them
3. Paste into the fields

### Operating Hours

Set hours for each day of the week:

- **Open/Close times**: 24-hour format
- **Closed checkbox**: Mark days when closed
- **Open 24 Hours**: For 24/7 establishments
- **Temporarily Closed**: For temporary closures

### Contact Information

- **Phone**: Primary phone number
- **Alt. Phone**: Secondary phone
- **Email**: Contact email
- **Website**: Full URL (https://...)
- **Facebook**: Facebook page URL
- **Instagram**: @username or full URL

### Photo Gallery

Add multiple photos:

1. Click "Add Images"
2. Select or upload images
3. Drag to reorder
4. Click × to remove

### Display Options

- **Featured Location**: Highlight on map with special marker
- **Marker Color**: Custom color for the map marker
- **Marker Icon**: Icon type (restaurant, cafe, hotel, etc.)

## Taxonomies

### Location Type

Hierarchical categories for classification:

- Food & Dining
  - Restaurant
  - Cafe
  - Bar
  - etc.
- Shopping
  - Mall
  - Boutique
  - etc.

### Location Tags

Non-hierarchical tags for features/amenities:

- wifi
- parking
- pet-friendly
- outdoor-seating
- etc.

### Area

Geographic areas/neighborhoods:

- Downtown / Poblacion
- Baybay Beach
- Pueblo de Panay
- etc.

### Price Range

Budget indicators:

- ₱ (Budget)
- ₱₱ (Moderate)
- ₱₱₱ (Upscale)
- ₱₱₱₱ (Fine Dining)

## Best Practices

1. **Always set coordinates** - Locations without coordinates won't appear on the map
2. **Use high-quality images** - At least 800×600 pixels
3. **Write descriptive content** - Help visitors understand what makes the place special
4. **Add relevant tags** - Makes filtering more useful
5. **Keep hours updated** - Especially for holidays and special events
